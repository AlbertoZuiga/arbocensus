\section{POSIBLES MEJORAS}

Entre las extensiones identificadas para versiones futuras del sistema se encuentran:

\begin{itemize}
  \item \textbf{Portal del censador}: interfaz \textit{mobile-first} para navegación en terreno con integración Google Maps y capacidad parcial de operación sin conexión.
  \item \textbf{Importación de formatos adicionales}: soporte GeoJSON e importación directa desde bases de datos legadas del sistema Arbocensus previo.
  \item \textbf{Matrices de costos de gran escala}: fragmentación (\textit{chunking}) de consultas OSRM para datasets que superen los 500 puntos activos.
  \item \textbf{Análisis de sensibilidad}: estudio sistemático de la relación entre el tiempo límite del solver y la calidad de la solución.
\end{itemize}
